Projects

1. Autonomous Driving Robot

This project aims to integrate autonomous driving robot using NVIDIA Jetson Nano board and Raspberry PI Camera. The purpose of this work is to train the NVIDIA Robot - which is a pre-built robot with a complete Artificial Intelligent (AI) development environment based on the tiny Jetson Nano computer chip, hardcoded using Python platform.

Therefore, the robot has the ability to avoid objects (collision avoidance) and keep track to drive safely within the lane lines (road following). Moreover, this autonomous robot can also follow the instruction signs and traffic signs such as a left turn, right turn, stop, and go to a pre-set destination.

This work uses PyTorch and Tensorflow to train the model and applies transfer learning to pre-trained deep learning architecture to optimize the system performance.

2. Home assessment of grasp development in infants at risk for fine motor delay

In this project, the purpose is to assess spontaneous hand use using video capture (Intel Realsense D435) and interactive grasp force with instrumented toys in infants. The Hand Use and Grasp Sensor (HUGS) system, which is a user-centered and home-based assessment tool, is designed to assess the development of hand grasp in infants from 12 - 36 weeks of age.

Toys will include biofeedback of the infantâ€™s own grasp force through different sensory channels, including touch (vibration), vision (lights), and hearing (sounds), to learn whether each infant is able to integrate sensory feedback into an adaptive hand grasp. This information could signal potential avenues for early interventions to encourage object exploration and functional hand use.

With home-based videos, different Deep Learning approaches used in implementation include: DeepLabCut, OpenPose, Deep High Resolution HRNet.
